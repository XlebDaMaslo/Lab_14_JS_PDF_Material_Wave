# Лабораторная работа №14. Скачивание файла в формате PDF. Добавление эффекта «Material Wave»

## Постановка задачи

Необходимо разработать веб-страницу с резюме, сверстанную по предоставленному шаблону. Страница должна обеспечивать возможность редактирования текстовых элементов, а также содержать кнопку «Скачать» для сохранения резюме в формате PDF. Дополнительно требуется реализовать анимацию взаимодействия пользователя с элементами страницы средствами CSS и добавить эффект «Material Wave» (ripple) [file:1].

## Основные требования

1. Разработать макет веб-страницы с резюме на основе предоставленного шаблона и выполнить fork исходного репозитория [file:1].
2. Реализовать возможность редактирования текстовых элементов резюме с сохранением изменений непосредственно внутри элементов [file:1].
3. Добавить кнопку «Скачать», которая позволяет сохранить текущую версию резюме в формате PDF на устройство пользователя [file:1].
   - Допускается использование библиотек для преобразования HTML в PDF, однако также допустим вариант с заранее подготовленным PDF-файлом и его скачиванием по ссылке [file:1].
4. Обеспечить анимацию изменений элементов резюме с помощью CSS [file:1].
5. Выполнить разработку с использованием базовых веб-технологий: HTML, CSS и JavaScript, без применения сторонних библиотек и фреймворков [file:1].

## Дополнительные требования

1. Обеспечить адаптивность интерфейса для корректного отображения страницы на различных устройствах и разрешениях экрана [file:1].
2. Реализовать эффект «Material Wave» (ripple) при взаимодействии пользователя с элементами страницы [file:1].
3. Обеспечить быструю загрузку страницы и сохранение введённых данных при её обновлении [file:1].

## Установка и запуск проекта

Для запуска проекта необходимо выполнить следующие действия [file:1].

1. Склонировать репозиторий:

```bash
git clone <URL_репозитория>
cd <имя_папки_проекта>
```

2. Установить зависимости:

```bash
npm install
```

## Команды для разработки

### Запуск локального сервера разработки

```bash
npm run dev
```

### Сборка проекта для продакшн

```bash
npm run build
```

После выполнения команды сборки итоговые файлы будут размещены в директории `/dist`. Именно эти файлы необходимо использовать для публикации проекта в продакшн-среде [file:1].
